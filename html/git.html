Git

  • git init - создание нового репозитория в Git в пустой директории или превращение существующей дериктории в новый репозиторий Git.
  • git clone "URL"- клонирование репозитория из удаленного хранилища на локальную машину.
  • git push - выгрузка сделанных изменений в удаленный репозиторий.
  • git pull - получение изменений из тудаленного репозитория на локальную машину.
  • git status - просмотр списак изменненых файлов.
  • git branch "название ветки" - создание ветки.
  • git checkout "название ветки" - переход на ветку.
  • git commit - сохранение изменеий в Git.
  • git log - просмотр истории коммитов в репозитории.
